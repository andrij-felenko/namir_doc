Qmi is a \char`\"{}$\ast$$\ast$\+Q$\ast$$\ast$t $\ast$$\ast$\+Mi$\ast$$\ast$nimal\char`\"{} theme for the Doxygen HTML documentation. It based on official Qt4 documentation\textquotesingle{}s style.

\doxysection*{How to setup}

To use {\ttfamily qmi} style make the following changes in your Doxyfile\+: \begin{DoxyVerb}# Project section
BRIEF_MEMBER_DESC = NO

# HTML section
HTML_HEADER = ${path_to_qmi}/header.html
HTML_FOOTER = ${path_to_qmi}/footer.html
HTML_STYLESHEET = ${path_to_qmi}/qmi.css
\end{DoxyVerb}
 {\bfseries{NOTE}}\+:


\begin{DoxyItemize}
\item If you use $\ast$$\ast$\+\_\+tree navigation panel\+\_\+$\ast$$\ast$ then copy contents of the {\ttfamily navtree} dir to the documentation html dir.
\item If you use $\ast$$\ast$\+\_\+search\+\_\+$\ast$$\ast$ feature then copy contents of the {\ttfamily search} dir to the {\ttfamily html/search}.
\end{DoxyItemize}
